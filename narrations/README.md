# Narrations

**Project purpose, values, and how to contribute** are in the repo root: **[`../README.md`](../README.md)** and **[`../INTENT_AND_VALUES.md`](../INTENT_AND_VALUES.md)**.

This directory holds **one narration at a time**. Each file is a mini-documentary script for an anti-school video.

**Workflow:** Create or open a single narration (e.g. one folder or one file per topic), complete it with research and citations, then start the next. Do not spread edits across multiple narrations in the same session.
