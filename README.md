# Anti-school

Scripts, research, and tooling for **mini-documentary narrations** that critique compulsory schooling—written for evidence-backed video, not debate-club both-sides framing.

## What’s here

- **`narrations/`** — Episode scripts (markdown), one focused topic at a time  
- **`research/`** — Notes and sources that feed the scripts  
- **`books/`** — Quote extraction pipeline and derived quote banks (see `books/extract_quotes.py`)  
- **`INTENT_AND_VALUES.md`** — Editorial stance and non-negotiable standards for anyone writing or editing here  

## Contribute

Pull requests and issues are welcome. Useful contributions include: **better citations** (peer-reviewed or solid gray literature), **fact-checks** with sources, **new narration outlines** or completed drafts, **research briefs**, and **small tooling fixes** (e.g. the quote extractor).

Please read **`INTENT_AND_VALUES.md`** first so your edits match the project’s purpose and citation rules. For narrations, keep changes **scoped to one episode per PR** when possible so review stays tractable.

If you open a PR, a short note on **what you changed** and **where the claims are sourced** helps a lot.

## License / use

Unless individual files say otherwise, treat content as **project-specific**; ask in an issue if you want to reuse large chunks elsewhere.
